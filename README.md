
# Decommit

**Decommit** is a decentralized version control system that combines IPFS-based storage with Ethereum smart contract governance. Unlike traditional Git, distribute voting rights, and approve changes entirely on-chain.

## How It Works

1. **Create a Project** – Start a new project
2. **Distribute Voting Tokens** – Allocate tokens to contributors for the current epoch.
3. **Submit Changes** – Contributors upload patches
5. **Vote on PRs** – Token holders vote to approve proposed changes.
6. **Merge Approved Changes** – Accepted PRs are recorded on-chain, ensuring immutable history.

## Getting Started

1. Install the local decommit client.
2. Connect your Ethereum wallet.
3. Create a project

**ChainForge** is a decentralized version control system that combines IPFS-based storage with Ethereum smart contract governance. Unlike traditional Git, ChainForge lets you manage projects, distribute voting rights, and approve changes entirely on-chain.

## How It Works

1. **Create a Project** – Deploy a new smart contract for your project.
2. **Distribute Voting Tokens** – Allocate tokens to contributors for the current epoch.
3. **Submit Changes** – Contributors upload patches or updates to IPFS.
4. **Vote on PRs** – Token holders vote to approve or reject proposed changes.
5. **Merge Approved Changes** – Accepted PRs are recorded on-chain, ensuring immutable history.
